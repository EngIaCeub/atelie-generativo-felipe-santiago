# Regras locais — resultados

- Armazene apenas artefatos pequenos, auditáveis e úteis ao relatório.
- Preferir CSV/JSON para dados e PNG/JPG para figuras finais.
- Cada resultado deve registrar configuração/seed/commit quando aplicável.
- Não editar manualmente métricas geradas por notebook; regenere a partir da fonte.
- Não versionar checkpoints ou pesos pesados; referenciá-los no manifesto.
- Dados de avaliação humana devem ser anonimizados.
